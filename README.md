# WhatsApp Clone App 📱

Yeh ek Android application hai jo WhatsApp ki core functionality (Messaging, Authentication) ko replicate karta hai. Ise Java aur Firebase ka use karke banaya gaya hai.

## 🚀 Features
- **Firebase Auth:** Secure Login aur Sign-up system.
- **Realtime Chat:** Firebase Realtime Database ka use karke instant messaging.
- **Modern UI:** XML layouts aur Material Design ka use.

## 🛠 Tech Stack
- **Language:** Java
- **UI:** XML (Android Studio)
- **Backend:** Firebase (Auth + Realtime Database)

## ⚙️ How to Run
1. Repo clone karo: `git clone [aapka-repo-link]`
2. Android Studio mein open karo.
3. Apna `google-services.json` file `app/` folder mein add karo.
4. Firebase console mein Authentication aur Database enable karo.
5. Run karke check karo!
