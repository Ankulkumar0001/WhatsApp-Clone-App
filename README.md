# WhatsApp Clone Application 📱

A fully functional WhatsApp Clone Android application, developed using Java and integrated with Firebase for backend services. This project replicates core messaging functionalities and serves as a portfolio project to demonstrate mobile app development skills.

## 🚀 Key Features
- **Firebase Authentication:** Secure user registration and login system.
- **Real-time Messaging:** Instant chat functionality powered by Firebase Realtime Database.
- **Modern UI:** Built with Android Studio using XML layouts and Material Design components for a seamless user experience.

## 🛠 Tech Stack
- **Language:** Java
- **Frontend:** XML, Material Design
- **Backend/Database:** Firebase (Authentication, Realtime Database)
- **IDE:** Android Studio
